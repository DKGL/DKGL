
You need prebuilt-binary (static-libraries) to build PyDK.

Download and extract prebuilt binaries here!

 Win32 (i386):
    https://github.com/tiff2766/Python/releases/download/v3.4.3/python3.4.3_Win32.zip
 Win32 (x64):
    https://github.com/tiff2766/Python/releases/download/v3.4.3/python3.4.3_Win32_x64.zip
 Mac OS X (universal):
    https://github.com/tiff2766/Python/releases/download/v3.4.3/python3.4.3_OSX.zip
 Apple iOS (universal):
    https://github.com/tiff2766/Python/releases/download/v3.4.3/python3.4.3_iOS.zip

or you can build from source at https://github.com/tiff2766/Python

directory structure for Xcode (iOS, OS X) is below
 ./iOS/libpython3.4.a
 ./OSX/libpython3.4.a

directory structure for Win32 is below
 ./Win32/python34_static_d.lib
 ./Win32/python34_static.lib
 ./Win32/python34.lib

directory structure for Win32_x64 is below
 ./Win32_x64/python34_static_d.lib
 ./Win32_x64/python34_static.lib
 ./Win32_x64/python34.lib
